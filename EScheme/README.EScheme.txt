Introduzione
------------
Questo è il client per il VDI, comunica con il sistema gexy per autenticare
l'utente e aprire il remote desktop

Comandi
-------
-V				Verbose, crea un file di log nella stessa directory
--vdname		prende come argomento il nome della vd da aprire senza
				chiedere all'utente
--configfile	Specifica il path del file di configurazione
